# 时迹 — 部署到 Cloudflare（小白版）

> 本项目是 **TanStack Start + Cloudflare Workers** 架构（不是普通静态站，所以 Netlify 默认配置会失败）。
> 推荐用 **Cloudflare Workers**（项目里已有 `wrangler.jsonc`，原生支持）。
> 部署后链接永久免费，全球 CDN，可分享给无限人，无需登录。

---

## 一、把代码推到 GitHub（一次性）

1. 在 Lovable 编辑器，点左下角聊天框的 **➕** → **GitHub** → **Connect project**
2. 授权 Lovable GitHub App
3. 选择你的 GitHub 账号 → 点 **Create Repository**
4. 完成后会得到仓库地址，类似 `github.com/你的名字/shiji`

> 🔒 代码现在永久存在 GitHub，Lovable 出任何问题都不影响。
> 建议把仓库设为 **Public**，这样即使账号丢失，别人也能 fork 继续部署。

---

## 二、注册 Cloudflare（免费）

1. 打开 https://dash.cloudflare.com/sign-up 用邮箱注册
2. 登录后，左侧菜单点 **Workers & Pages**

---

## 三、连接 GitHub 部署到 Cloudflare Workers

1. 点 **Create** → 选 **Workers** 标签 → **Import a repository**（或 **Connect to Git**）
2. 授权 Cloudflare 访问 GitHub，选择 `shiji` 仓库
3. 在 **Build configuration** 页面填写：

   | 字段 | 填写内容 |
   |---|---|
   | Build command | `bun run build` |
   | Deploy command | `npx wrangler deploy` |
   | Root directory | （留空） |

4. 点 **Save and Deploy**，等待 2–5 分钟构建完成

---

## 四、拿到永久链接

构建成功后会得到一个网址，类似：

**`https://tanstack-start-app.你的用户名.workers.dev`**

把这个链接发给任何人，打开就能用：
- ✅ 无需登录
- ✅ 永久免费
- ✅ 可无限分享
- ✅ 数据存在每个人自己的浏览器里，互不干扰

---

## 五、之后怎么更新

在 Lovable 改代码 → 自动同步到 GitHub → Cloudflare Workers 自动重新部署。
**链接永远不变，全程零操作。**

---

## 六、（可选）绑定自己的域名

如果你买了域名（例如 `shiji.com`）：

1. Cloudflare Dashboard → 你的 Worker → **Settings** → **Domains & Routes**
2. 点 **Add → Custom Domain**，输入域名
3. 按提示在域名注册商修改 DNS（Cloudflare 会自动配置 SSL）

---

## 七、双保险：让账号丢失也不怕

1. **GitHub 仓库设为 Public**：账号被封别人也能 fork 继续部署
2. **定期点 App 内"导出备份"**：JSON 文件存云盘/邮箱，换设备一键导入即可恢复全部数据

---

## 八、卡住怎么办

- 构建失败：把 Cloudflare 的构建日志截图发给 Lovable AI
- 链接打不开：等 5 分钟，DNS 需要传播时间
- 数据没了：用之前导出的 JSON 备份文件，在设置页"导入备份"
